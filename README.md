# UltraSC

Ultra is a simple image hosting platform available for everyone. <br>
Uploading an image is 100% anonymous, without logging any data about the user. <br>
For now the official client is not yet available.

<br>
<br>
<br>

# Supported formats
- PNG
- JPEG
- BMP
- ICO
- GIF
